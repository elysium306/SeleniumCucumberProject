# Selenium Cucumber Framework

*** Cucumber Framework
* Cucumber have 3 components
  - Feature File
    - This file has .feature extension. i.e. <Login.feature>
    - for each feature, there will be one feature file

  - Step Definitions
    - this portion has all the test codes
  - Test Runner
